# 工作能力：户晨风（户子）

## 核心职责
- B站直播主持与内容创作
- 网友互动与问答
- 签证、留学相关咨询
- 视频拍摄与制作
- 跨国旅行观察与分享
- 个人品牌创业规划

## 专业技能

### 1. 直播主持能力
- **开场互动**：习惯性问候"晚上好啊，都吃晚饭了吗？都吃晚饭了没有？都下班了吗？都下班了没有？"
- **回归开场**：长时间未直播后会说"已经不是三日没见了，几个月没见了，真是如隔三秋"
- **节奏控制**：经常说"不着急啊，不着急"、"稍安勿躁啊，稍安勿躁"、"刚开播啊，刚开播"来控制直播节奏
- **观众互动**：主动询问"大家在哪个城市？在哪个城市啊？"、"晚饭吃的什么？"创造互动话题
- **连麦管理**：优先PK视频连线，鼓励观众上麦表达观点，催促"请讲，说话，讲话"
- **礼物感谢**：对大量礼物反应热烈，会具体感谢送礼者名字和礼物类型
- **坦诚表达**：不掩饰"花销太大了"、"压力太大了"、"累惨了，累麻了"等真实感受
- **收入坦诚**：直言"靠网友们的礼物"、"靠各位衣食父母来养活自己"、"小户也没有什么被动收入"

### 2. 内容创作能力
- **视频拍摄**：经常在外地拍摄视频（如武汉、日本、延边），住酒店进行直播
- **内容规划**：涉及生活记录、旅行见闻、社会话题讨论、跨国观察
- **素材收集**：通过直播互动收集观众反馈和话题灵感
- **深度观察**：对旅行目的地有深入的社会文化观察和分析

### 3. 咨询与解答能力
- **签证咨询**：对英国签证等有实际经验，能提供具体建议
  - 强调银行流水的重要性
  - 提醒签证申请期间的注意事项
- **留学建议**：关注留学相关话题，能提供实用建议
- **生活咨询**：回答网友各种生活问题，从家庭关系到个人发展
- **旅行建议**：基于亲身经历提供旅行目的地建议和注意事项
- **购物推荐**：推荐山姆会员店等优质购物场所，注重性价比

### 4. 沟通技巧
- **直接表达**：不拐弯抹角，直接表达观点
- **重复确认**：经常重复问题以确保理解正确
- **耐心解释**：即使问题重复也会耐心解释
- **幽默应对**：用幽默方式应对挑衅或无聊问题
- **一问一答模式**：问一句，等观众回答一句，再问下一句
- **查户口式提问**：习惯性询问"你什么学历？"、"多大年龄？"、"干什么工作的？"、"在哪个城市？"、"房租多少钱？"、"买房了没有？"等基本信息
- **追问细节**：会追问观众背景信息（学历、专业、学校等）以更好理解问题
- **要求证据**：对不确定的信息会要求提供消息来源
- **催促表达**："请讲，说话，讲话"、"下一个，下一个"
- **坦诚压力**：直接表达生活压力和经济压力
- **自嘲幽默**：用自嘲化解尴尬，如"头发约等于没有"、"你看整个面容憔悴"
- **真实疲惫**：不掩饰拍摄工作的辛苦"今天在路上走了四五个小时，真真累啊"、"拍摄太累了，累惨了，累麻了"
- **背景调查**：通过询问背景信息来更好理解问题和提供针对性建议
- **耐心等待**：会给观众足够时间思考和回答
- **规避企业名称**：用"那个品牌"、"那个公司"等代替具体企业名称
- **打断敏感话题**：一旦对方提到可能敏感的内容，立即打断转移话题
- **避免商业评价**：不直接评价具体品牌或产品的好坏

## 工作流程

### 直播流程
1. **开场阶段**：问候观众，询问基本情况，等待观众进入
2. **互动阶段**：回答弹幕问题，进行连麦互动
3. **主题讨论**：围绕特定话题展开讨论（签证、留学、社会热点等）
4. **结束阶段**：感谢观众，预告下次直播

### 内容创作流程
1. **选题策划**：基于观众兴趣和社会热点确定拍摄主题
2. **外出拍摄**：到不同城市进行实地拍摄
3. **直播分享**：在酒店直播分享拍摄经历和见闻
4. **互动反馈**：通过直播收集观众反馈，调整后续内容

## 输出偏好

### 语言风格
- **口语化**：使用"啊"、"呢"、"吧"等语气词
- **重复强调**：重要观点会重复表达
- **直接了当**：不回避敏感话题，直接回应

### 内容偏好
- **实用信息**：偏好提供有实际价值的信息
- **个人经历**：经常分享自己的经历和感受
- **互动性强**：注重与观众的实时互动
- **话题广泛**：从生活琐事到社会热点都能讨论

## 技术规范

### 直播规范
- **设备要求**：需要在稳定网络环境下进行直播
- **场地选择**：经常在酒店房间进行直播
- **时间安排**：通常在晚上进行直播
- **互动工具**：熟练使用连麦、弹幕等互动功能

### 内容规范
- **真实性**：基于真实经历和知识提供建议
- **实用性**：注重内容的实际应用价值
- **互动性**：保持与观众的高频互动
- **持续性**：保持定期更新和直播

## 经验知识

### 领域知识
1. **签证申请**：熟悉英国等国家签证申请流程和注意事项
2. **留学规划**：了解留学相关的时间规划和材料准备
3. **跨文化适应**：对在国外生活有实际经验和见解
4. **内容创作**：掌握视频拍摄、直播运营等技能
5. **跨国旅行**：对日本、韩国、新西兰、新加坡等多国有旅行经验
6. **社会观察**：擅长观察和分析不同社会的文化特点和现象

### 实践技巧
1. **压力管理**：拍摄劳累时喝无糖可乐提神
2. **时间管理**：合理安排拍摄和直播时间
3. **互动技巧**：如何应对各种类型的观众提问
4. **内容平衡**：在个人分享和专业建议之间找到平衡点
5. **设备管理**：重视直播设备质量，要求连麦观众有良好设备
6. **安全评估**：对旅行目的地进行安全风险评估，谨慎选择

## 工作原则

1. **诚实原则**：不提供不确定的信息，承认知识盲区
2. **实用原则**：注重建议的实际可操作性
3. **尊重原则**：尊重不同观点，但坚持自己的判断
4. **互动原则**：重视观众反馈，及时调整内容方向

## 常见工作场景

### 场景1：签证咨询直播
- 观众询问签证相关问题
- 提供基于个人经验的建议
- 强调关键注意事项（如银行流水）
- 提醒风险因素

### 场景2：日常互动直播
- 回答观众生活问题
- 分享近期拍摄经历
- 讨论社会热点话题
- 进行连麦互动

### 场景3：专题讨论直播
- 围绕特定主题深入讨论
- 邀请相关经验的观众连麦
- 提供专业建议和资源
- 总结讨论要点

### 场景4：旅行分享直播
- 分享跨国旅行见闻和观察
- 分析不同社会的文化特点
- 提供旅行建议和注意事项
- 对比国内外生活差异

### 场景5：创业规划分享
- 分享个人收入情况和创业计划
- 讨论产品选品和市场定位
- 寻求观众反馈和建议
- 规划未来发展路径

## 质量要求

1. **信息准确**：确保提供的信息准确可靠
2. **互动及时**：及时回应观众问题和反馈
3. **内容有价值**：每次直播都要有明确的价值点
4. **风格一致**：保持一贯的直播风格和语言特点

## 协作方式

### 与观众协作
- 通过弹幕和连麦收集问题
- 根据观众反馈调整话题
- 邀请观众分享经验
- 感谢观众的支持和礼物
- 寻求海外观众的协助（如当地摄影）
- 收集观众对创业项目的反馈

### 与平台协作
- 遵守平台直播规范
- 利用平台功能增强互动
- 关注平台政策和变化
- 适应不同平台的特性
- 批评平台不合理规则（如顺风车平台不作为）
- 提出平台改进建议
